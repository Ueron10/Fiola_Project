RINGKASAN ALUR KERJA APLIKASI
==============================

1. USER BUKA APLIKASI (http://127.0.0.1:5000)
   - Flask route '/' render index.html

2. PILIH METODE INPUT:
   a) Upload Foto: Drag & drop atau pilih file
   b) Capture Foto: Buka kamera, ambil foto tunggal
   c) Live Camera Real-time: Buka kamera dengan deteksi berkelanjutan

3. UPLOAD FOTO / CAPTURE FOTO:
   - File/foto disimpan di selectedFile
   - Tampilkan preview
   - User klik "Prediksi Foundation Shade"
   - Kirim ke backend via POST /predict
   - Backend proses gambar dengan YOLO
   - Return hasil prediksi
   - Tampilkan hasil dengan confidence bars

4. LIVE CAMERA REAL-TIME:
   - JavaScript request akses kamera
   - Video stream ditampilkan di videoElement
   - COCO-SSD model dimuat untuk deteksi wajah
   - Setiap 1 detik, deteksi persons di video frame
   - Bounding box hijau digambar di canvas
   - Jika wajah terdeteksi dan tidak sedang prediksi (rate limiting):
     - Capture frame ke canvas
     - Konversi ke base64
     - Kirim ke backend via POST /predict_frame
   - Backend decode frame dan proses dengan YOLO
   - Return hasil prediksi
   - Tampilkan hasil di overlay video (Foundation: XXX, Skin: XXX (XX%))
   - Rate limiting mencegah multiple request simultaneously

5. BACKEND PROCESSING (/predict dan /predict_frame):
   - Decode gambar/frame ke OpenCV format
   - Load YOLO model
   - Detect wajah dengan YOLO
   - Crop region wajah
   - Analisis skin tone (RGB + HSV analysis)
   - Klasifikasi ke Fair/Medium/Tan
   - Hitung confidence score
   - Map ke foundation shade
   - Return JSON dengan hasil prediksi

6. FITUR RATE LIMITING:
   - Flag isPredicting mencegah multiple prediksi simultaneously
   - Hanya satu prediksi berjalan dalam satu waktu
   - Mengurangi beban server dan mencegah error
   - Status message menunjukkan state prediksi
